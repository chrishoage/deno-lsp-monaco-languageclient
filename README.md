# Deno LSP + monaco-languageclient

This project serves to demonstrate using Monaco with the Deno LSP

## Prerequisites

- Deno
- pnpm

## Running

### Start the server

1. `cd server`
2. `deno -A server.ts`

### Start the client

1. `cd client`
2. `pnpm install`
3. `pnpm start`

### Open the browser

1. Navigate to http://localhost:5173/
